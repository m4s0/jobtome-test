# Application

Application runs with PHP 8 and MySQL 8

#### Docker

Build container

```
make build
```

Run container

```
make up
```

Stop container

```
make down
```

Enter into php container

```
make bash
```

#### Init Application

install dependencies

```
make install
``` 

#### Tests

```
make test
``` 

#### Coding Standards

```
make cs
``` 

#### Static Code Analysis

```
make stan
```
